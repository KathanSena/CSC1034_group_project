Data rules

Validation checks

  • Household size, postcode, and referral source must all be completed.
  • Household size must be a number between 1 and 20.
  • Postcode must be between 5 and 20 characters after trimming spaces.
  • The submit button is disabled while a save is in progress.
  • Delete actions show a confirmation dialogue before the SQL runs.
